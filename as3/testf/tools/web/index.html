Welcome to TestF

TestF is an Open Source benchmark for Flash devices running Flash Player 10.1, AIR 2.5 or higher.

Its aim is putting Flash devices to the test, highlighting strengths and weaknesess. Which one has more raw CPU power. Which one has better access to disk. Which one has better hardware acceleration. And so on.

You can run the test now, but please read below some considerations:

  • Close all other applications and background processes first, you don't want other apps interfering the benchmark.
  • Connect your device to a power source, many turn down CPU cycles when on battery.
  • The test will deliberately push the device to its limit, so expect some screen tearing, freezes, fans spinning, etc.
  • There are big differences between running Flash apps within AIR vs. standalone vs. within the browser.
  • There is a huge difference between using a debug vs. release version of the player.

TestF is Open Source, you can get the source code in Github. Anybody is more than welcome to provide new tests or improve the current ones, just have a read to the docs.
